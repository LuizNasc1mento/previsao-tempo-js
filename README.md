# 🌤️ Weather Forecast App

Um aplicativo de previsão do tempo moderno e responsivo que utiliza a API do OpenWeatherMap para fornecer dados em tempo real com uma experiência de usuário fluida e intuitiva.

## 🚀 Funcionalidades Principais

- *Busca Global:* Pesquisa de clima por nome da cidade com suporte a códigos de país.
- *📍 Geolocalização:* Botão dedicado para obter o clima local instantaneamente via GPS do navegador.
- *💾 Persistência de Dados (LocalStorage):* O app memoriza a última cidade pesquisada, carregando os dados automaticamente ao abrir o site.
- *🎨 Design Glassmorphism:* Interface moderna com efeitos de transparência, desfoque (backdrop-filter) e sombras suaves.
- *🌈 Fundo Dinâmico:* As cores do background se adaptam automaticamente à condição climática (Sol, Nuvens, Chuva).

## 🛠️ Tecnologias Utilizadas

- *JavaScript (ES6+):* Fetch API, Async/Await, manipulação de DOM e LocalStorage.
- *HTML5 & CSS3:* Uso de CSS Grid, Flexbox e variáveis para um design limpo.
- *OpenWeather API:* Fonte de dados meteorológicos em tempo real.
- *Font Awesome:* Ícones vetoriais para interface.
